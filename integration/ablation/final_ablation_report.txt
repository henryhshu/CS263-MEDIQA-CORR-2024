KEY FINDINGS
====================================================================================================

1. MULTI-AGENT IS THE ESSENTIAL BACKBONE
   Multi-Agent alone outperforms both RAG-only and ICL-only by a large margin
   (AggScore 0.745 vs 0.589 / 0.660). Without MA, combining RAG+ICL still only
   reaches AggScore 0.744 — essentially the same as MA alone. The structured
   Detector→Critic→Editor→Critic pipeline is the strongest single component.

2. RAG MOST BENEFITS CORRECTION QUALITY (text metrics)
   RAG + MA (no ICL) achieves the best raw text quality scores:
     ROUGE-1 0.773  ROUGE-L 0.768  BERTScore 0.793  AggScore 0.783
   Adding drug reference context to the Detector and Critic grounds their
   corrections in verified pharmacological facts, producing more accurate
   sentence rewrites. This is the strongest pairwise combination for NLG metrics.

3. ICL MOST BENEFITS ERROR DETECTION (accuracy metrics)
   ICL + MA (no RAG) achieves the best pairwise error detection:
     Flag Accuracy 0.84  Sentence Accuracy 0.82
   In-context examples calibrate the model on what a valid error looks like,
   reducing false positives and improving localization before the editor runs.

4. FULL COMBINATION WINS ON DETECTION + COMPOSITE METRICS
   RAG + ICL + MA achieves the best overall detection (FlagAcc 0.86,
   SentAcc 0.84) and composite scores (AggComposite 0.743, BERTScr Comp 0.756,
   BLEURT Comp 0.609), confirming that all three components are complementary:
   RAG grounds corrections, ICL improves detection, MA structures reasoning.

5. BLEURT REVEALS A TRADE-OFF
   BLEURT scores tell a different story than ROUGE/BERTScore: RAG + ICL (no MA)
   scores highest on raw BLEURT (0.532), followed by RAG + MA (0.508) and MA
   alone (0.520). The full combined system scores lower on raw BLEURT (0.455),
   suggesting the multi-agent critic sometimes makes conservative edits that
   differ more from the reference phrasing even when factually correct.
   BLEURT Composite (which credits correct NA predictions) reverses this —
   full combined scores best (0.609) — indicating the detection accuracy
   advantage outweighs the phrasing difference for composite scoring.

6. RAG ALONE IS INSUFFICIENT
   RAG-only has the worst detection accuracy (FlagAcc 0.76, SentAcc 0.72) and
   BLEURT (0.260). Drug context augmentation alone without structured reasoning
   or task demonstrations does not meaningfully improve over a simple prompt.

7. RECOMMENDED CONFIGURATION
   For best overall performance: RAG + ICL + MA (full combined).
   For best correction text quality: RAG + MA (skip ICL to reduce prompt length).
   For best detection only: ICL + MA (skip RAG if drug content is sparse).
   These three configurations are all within ~0.04 AggScore of each other,
   so the choice can be guided by latency or cost constraints.

NOTE: All results are on the 50-sample MEDEC-MS test subset. Findings should
be validated on the full 597-sample test set before drawing final conclusions.
